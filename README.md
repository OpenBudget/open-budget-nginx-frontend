# Open Budget Nginx Frontend

## Using docker-compose to start the full budgetkey docker environment

**WARNING** this environment is extremely resource intensive, start with caution.

* Install Docker and Docker Compose
* Start the environment
```
docker-compose up -d
```
* replace any next.obudget.org url with localhost
  * http://localhost/
  * http://localhost/pipelines/
  * http://localhost/app/search
  * etc..
